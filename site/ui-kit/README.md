# UI Kit — Ваш Финдир

Дизайн-система проекта. Источник истины для всех экранов.

## Файлы

- `storybook.html` — визуальный референс. Открой в браузере.
- `decisions.md` — текстовые правила и токены. Читается за 5 минут.
- `design-audit.md` — исходный аудит, откуда всё началось.

## Как пользоваться

1. Перед новым экраном — открой `storybook.html`, найди нужные компоненты.
2. Скопируй HTML-сниппет из `storybook.html` или попроси Claude — он знает.
3. Все цвета/размеры — только через CSS-переменные из `:root` storybook.
4. Money — строго по правилам секции 04 storybook (или раздел Money в decisions.md).

## Что НЕ делать

- Не создавай произвольные цвета/размеры в экранах.
- Не делай отдельные пикеры — используй EntityPicker / TreePicker.
- Не раскрашивай иконки в меню. Иконки = currentColor.
- Новые компоненты — только через UI Kit, не в экранах.
