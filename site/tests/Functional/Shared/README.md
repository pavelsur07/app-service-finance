# Functional/Shared

Общее для функциональных тестов (MVP).

- Builders-first: сценарии опираются на builders.
- Слои тестирования: Unit, Integration, Functional.
- Функциональные тесты идут через реальные границы приложения.
- Явные операции с БД: persist/flush/clear.
- Моки в домене запрещены.
- Общие хелперы кладём сюда, инфраструктуру — в tests/_support.
- Модули пока отсутствуют.
- Держим helpers понятными и устойчивыми.
