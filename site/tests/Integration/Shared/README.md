# Integration/Shared

Общее для интеграционных тестов (MVP).

- Builders-first: используем builders как первичный способ сборки данных.
- Слои тестирования: Unit, Integration, Functional.
- Интеграция работает с реальными зависимостями.
- Явные операции с БД: persist/flush/clear.
- Моки в домене запрещены.
- Общие хелперы храним здесь, инфраструктуру — в tests/_support.
- Модули пока не заведены.
- Держим helpers переиспользуемыми.
